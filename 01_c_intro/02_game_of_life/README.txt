Реализация game of life

--------------------------------------------------------------------------------
Для начала, гуглим game of life, читаем.

Вкратце - есть некоторое поле с ячейками, матрица of sorts.
Ячейка может быть занята живой клеткой, или мёртвой.

Правила такие:
  - любая живая клетка, у которой меньше двух живых соседних клеток - умирает.
  - любая живая клетка, у которой больше трёх живых соседних клеток - умирает.
  - любая живая клетка, у которой 2 или 3 соседа - остаётся живой.
  - любая мёртвая клетка с тремя живыми соседями - оживает.

Поле с ячейками можно сделать статичным - 80 х 25, при желании можно реализовать масштабируемость.

Пускай программа принимает в stdin файлик 80 x 25 с изначальным состоянием, а затем запускает симуляцию.
Либо же можно сделать рандомную генерацию поля при запуске.

Также можно добавить регулируюмую скорость симуляции.
